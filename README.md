# cashback-module-public

Public changelog viewer for the cashback rewards platform.

## Files

| File | Description |
|------|-------------|
| `index.html` | Single-page viewer — HTML, CSS, JS inline |
| `CHANGELOG.md` | Ukrainian changelog |
| `CHANGELOG.en.md` | English translation |
| `images/changelog/` | Changelog images/GIFs |

> Content is managed and synced automatically via CI. Do not edit files directly in this repo.
